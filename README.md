## 📖 Project Overview

**Legacy of Dreams** is a modern tribute website dedicated to **Dr. A. P. J. Abdul Kalam**, India's former President, aerospace scientist, and one of the greatest inspirations for students.

The project is developed as part of the **Oasis Infobyte Web Development & Designing Internship (OIBSIP)**.

The website combines beautiful UI design, responsive layouts, glassmorphism, smooth animations, and interactive JavaScript to provide an engaging experience.
